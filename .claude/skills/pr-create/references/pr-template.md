# PR 説明文テンプレート

```markdown
## Summary

変更内容の概要を 1-2 行で説明

## Changes

- 変更内容 1
- 変更内容 2

## Type of Change

- [ ] New Feature
- [ ] Bug Fix
- [ ] Refactoring
- [ ] Documentation
- [ ] Performance Improvement

## Related Issues

Closes #123（関連 Issue がある場合）
```

## タイプのマッピング

Conventional Commits の type から判定：

| type       | Type of Change          |
| ---------- | ----------------------- |
| `feat`     | New Feature             |
| `fix`      | Bug Fix                 |
| `refactor` | Refactoring             |
| `docs`     | Documentation           |
| `perf`     | Performance Improvement |
